# Alura Challenge 5 º Edição

## Server
Crie um arquivo .env seguindo o modelo .env.example na pasta server. Preenchendo os dados para o banco de dados.
```
DB_USER=Joao
DB_PASSWORD=senha
```
Instale as dependências
```
npm i
```
Iniciar o servidor
```
npm start
```

## Web
Instale as dependências
```
yarn i
```
Inicie a aplicação
```
yarn start
```
